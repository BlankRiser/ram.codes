### Why are forms essential?


Forms are the most common way to collect data from users. They are used for login, registration, checkout, and more. Forms need to be easy to use and accessible to all users.

### Using FormData API

```tsx title="Using form data api" 
export const ReactFormData = () => {
  const handleSubmit = (e: FormEvent<HTMLFormElement>) => {
    e.preventDefault();

    const formData = new FormData(e.currentTarget);
    const values = Object.fromEntries(formData.entries());
    console.log(values);
  };

  return (
    <form onSubmit={handleSubmit}>
      <label htmlFor='username'>Username</label>
      <input name='username' id='username' placeholder='username' type='text' />
      <label htmlFor='password'>Password</label>
      <input name='password' id='password' placeholder='password' type='text' />
      <button type='submit'>Submit</button>
    </form>
  );
};
```

Here is a simple form that collects a username and password. When the form is submitted, the `handleSubmit` function is called. The `FormData` object is created from the form element. The `entries` method returns an iterator of key/value pairs. The `Object.fromEntries` method converts the iterator to an object.
